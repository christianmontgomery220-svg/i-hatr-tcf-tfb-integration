# I-HATR Integration — TCF/TFB + Anonymo Framework

This repository contains a technical note and supplementary materials related to the integration of a Human–AI trust calibration perspective (I-HATR) into the Anonymo AI Framework (24×24), based on the TCF/TFB conceptual foundation.

## Scope

This repository does not define a new framework and does not modify the core architecture of the Anonymo AI Framework.

It should be understood as a supplementary technical layer associated with the consolidated v2.0 release of the framework.

## Relationship to Framework v2.0

The material presented here operates transversally across the existing architecture:

- Layer -1 (transparency and governed entry)
- A0 (triage)
- Governed interaction layers (A1–A4)

It contributes to:
- trust calibration
- uncertainty disclosure
- behavioral governance of AI interaction
